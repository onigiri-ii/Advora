# Advora

Introducing Advora, your healthcare advocate, right in your pocket! 

To run the envirnoment make a virtual envirnoment in pything version 3.11

```
py -3.11 -m venv venvName
```

Then, after activating the virtual envirnoment, download the requirements using
```
pip install -r requirements.txt
```

Make sure to make a .env file to to update your api!
