# Advora

Introducing Advora, your personal healthcare advocate, right in your pocket! 

To run the envirnoment make a virtual environment in Python version 3.11

```
py -3.11 -m venv venvName
```

Then, after activating the virtual environment, download the requirements using
```
pip install -r requirements.txt
```

Make sure to make a .env file to update your api!
